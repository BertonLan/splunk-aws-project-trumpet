Splunk AWS logging Setup

Use the following form to create a unique CloudFormation template that will forward logs from your AWS environment to Splunk. Once you have completed the form, deploy the CloudFormation template using the AWS CLI or AWS CloudFormation console.

For each AWS service you select, you will need a new enabled HEC token.
